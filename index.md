**常见问题**
- **如何启用键盘（iOS）？** 设置 → 通用 → 键盘 → 键盘 → 添加新键盘 → 选择「写作输入法」；如需完整功能，请在该键盘下打开「允许完全访问」。
- **为什么需要「完全访问」？** 仅用于读取内置码表、保存你的自建词库与候选排序（数据都存在设备本地）。本 App 不联网、不上传任何内容。
- **如何安装（macOS）？** 将 App 放入 `~/资源库/输入法/`，然后在 系统设置 → 键盘 → 文本输入 → 输入法 中添加「写作输入法」。
- **数据会不会丢？** 你的自建码表、词库、设置都保存在本地，可通过导入/导出功能备份与迁移。

### Support (English)

Thank you for using **WritingIME**, a minimal, fast, fully-offline Wubi (五笔) input method.

- Email: yyi987558@gmail.com (reply within 1–2 business days)
- **Enable keyboard (iOS):** Settings → General → Keyboard → Keyboards → Add New Keyboard → WritingIME. For full functionality, enable "Allow Full Access".
- **Why Full Access?** Only to read built-in code tables and save your custom dictionaries and candidate ordering locally. The app has no network access and uploads nothing.
- **Install (macOS):** Put the app in `~/Library/Input Methods/`, then add "写作输入法" in System Settings → Keyboard → Text Input → Input Sources.

### Privacy Policy (English)

Last updated: July 2, 2026

**WritingIME** ("the App") respects your privacy and is designed to run **entirely on your device**.

**Information we collect: none.** The App does not collect, store, upload, or share any personal information or usage data. We operate no servers and share no data with any third party.

**What you type** is processed locally on your device only, solely to generate candidates and insert text. It is never logged, saved, or transmitted anywhere.

**Your custom data** (code tables, dictionaries, candidate ordering, settings) is stored only locally on your device (an App Group container on iOS; the Application Support directory on macOS), so the main app and keyboard can share it. You can back it up via import/export, and remove it by deleting the App.

**About "Full Access" (iOS keyboard):** Due to iOS sandbox rules for third-party keyboards, the keyboard can only read/write the local container shared with the main app when Full Access is enabled. This permission is requested solely for that local data sharing, never for network transmission — the App contains no networking code and does not collect or send your keystrokes.

**Network:** The App requests no network access and runs fully offline.

**Third parties:** The App contains no third-party analytics, advertising, or tracking SDKs.
